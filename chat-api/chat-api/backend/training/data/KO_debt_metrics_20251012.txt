### Key Notes & Methodology:
- **Total Debt** = Long-term debt + Short-term debt/current portion[1][2][4][6].
- **Shareholders' Equity, Assets, Revenues, EBIT, Interest Expense, Net Income, Cash, Marketable Securities**: Extracted from balance sheet, income statement, and cash flow statement[1][2][4][6].
- **Operating Cash Flow (OCF)** and **CapEx** taken from cash flow statement; **Free Cash Flow** calculated as OCF – CapEx[1][2][4][6].
- **Ratios:**
  - Debt-to-Equity = Total Debt / Shareholders' Equity.
  - Interest Coverage = EBIT / Interest Expense.
  - Net Debt = Total Debt - Cash & Equivalents.
  - ROE = Net Income / Shareholders' Equity.
  - ROA = Net Income / Total Assets.

- **Fiscal Year End Date:** December 31 for all years[1][2][4][6].
- **Marketable Securities**: Values shown only if separately reported.
- If a row or value is not reported, it is considered zero or omitted per SEC standards.
