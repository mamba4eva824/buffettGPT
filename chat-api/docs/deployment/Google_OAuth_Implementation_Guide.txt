# Google OAuth Integration Implementation Guide
# Financial Chatbot Backend - Phase 1 Authentication

## Project Overview

### Goal
Implement Google OAuth 2.0 authentication for the Warren Buffett Financial Advisor chatbot, replacing complex AWS Cognito setup with a simpler, faster-to-market solution while maintaining enterprise-grade security.

### Architecture Decision
- Use Google OAuth for user authentication (no AWS Cognito initially)
- AWS API Gateway with custom JWT authorizer
- DynamoDB for user profiles and session management
- Lambda functions for token verification and user management

## Phase C: Lambda Function Implementation

### C.1 Google OAuth Verification Lambda

#### File: lambda-functions/auth-verify/lambda_function.py
```python
import json
import os
import jwt
import uuid
import time
from datetime import datetime, timedelta
import boto3
from google.auth.transport import requests
from google.oauth2 import id_token
import logging

# Configure logging
logger = logging.getLogger()
logger.setLevel(logging.INFO)

# Initialize AWS clients
dynamodb = boto3.resource('dynamodb')
users_table = dynamodb.Table(os.environ['USERS_TABLE'])
sessions_table = dynamodb.Table(os.environ['SESSIONS_TABLE'])

# Environment variables
GOOGLE_CLIENT_ID = os.environ['GOOGLE_CLIENT_ID']
JWT_SECRET = os.environ['JWT_SECRET']

def lambda_handler(event, context):
    """
    Main Lambda handler for Google OAuth verification
    Supports both API Gateway requests and Custom Authorizer requests
    """
    try:
        # Check if this is a custom authorizer request
        if 'type' in event and event['type'] == 'TOKEN':
            return handle_authorizer_request(event)
        
        # Handle API Gateway request
        return handle_api_request(event, context)
        
    except Exception as e:
        logger.error(f"Error in lambda_handler: {str(e)}")
        return {
            'statusCode': 500,
            'headers': {
                'Access-Control-Allow-Origin': '*',
                'Access-Control-Allow-Headers': 'Content-Type,Authorization',
                'Access-Control-Allow-Methods': 'OPTIONS,POST'
            },
            'body': json.dumps({'error': 'Internal server error'})
        }

def handle_api_request(event, context):
    """Handle direct API Gateway requests for authentication"""
    
    # Handle CORS preflight
    if event['httpMethod'] == 'OPTIONS':
        return {
            'statusCode': 200,
            'headers': {
                'Access-Control-Allow-Origin': '*',
                'Access-Control-Allow-Headers': 'Content-Type,Authorization',
                'Access-Control-Allow-Methods': 'OPTIONS,POST'
            },
            'body': ''
        }
    
    if event['httpMethod'] != 'POST':
        return {
            'statusCode': 405,
            'headers': {'Access-Control-Allow-Origin': '*'},
            'body': json.dumps({'error': 'Method not allowed'})
        }
    
    try:
        # Parse request body
        body = json.loads(event['body'])
        google_token = body.get('token')
        
        if not google_token:
            return {
                'statusCode': 400,
                'headers': {'Access-Control-Allow-Origin': '*'},
                'body': json.dumps({'error': 'Google token is required'})
            }
        
        # Verify Google token
        user_info = verify_google_token(google_token)
        if not user_info:
            return {
                'statusCode': 401,
                'headers': {'Access-Control-Allow-Origin': '*'},
                'body': json.dumps({'error': 'Invalid Google token'})
            }
        
        # Create or update user
        user_data = create_or_update_user(user_info)
        
        # Create session
        session_data = create_user_session(user_data['user_id'])
        
        # Generate JWT token
        jwt_token = generate_jwt_token(user_data['user_id'], session_data['session_id'])
        
        return {
            'statusCode': 200,
            'headers': {'Access-Control-Allow-Origin': '*'},
            'body': json.dumps({
                'token': jwt_token,
                'user': {
                    'id': user_data['user_id'],
                    'email': user_data['email'],
                    'name': user_data['name'],
                    'picture': user_data.get('picture', ''),
                    'subscription_tier': user_data.get('subscription_tier', 'free')
                },
                'session': {
                    'id': session_data['session_id'],
                    'expires_at': session_data['expires_at']
                }
            })
        }
        
    except json.JSONDecodeError:
        return {
            'statusCode': 400,
            'headers': {'Access-Control-Allow-Origin': '*'},
            'body': json.dumps({'error': 'Invalid JSON in request body'})
        }

def handle_authorizer_request(event):
    """Handle API Gateway Custom Authorizer requests"""
    
    token = event['authorizationToken']
    
    # Remove 'Bearer ' prefix if present
    if token.startswith('Bearer '):
        token = token[7:]
    
    try:
        # Verify JWT token
        payload = jwt.decode(token, JWT_SECRET, algorithms=['HS256'])
        user_id = payload['user_id']
        session_id = payload['session_id']
        
        # Verify session is still valid
        session = verify_session(session_id, user_id)
        if not session:
            raise Exception('Session expired or invalid')
        
        # Generate policy
        policy = generate_policy(user_id, 'Allow', event['methodArn'])
        
        # Add user context
        policy['context'] = {
            'user_id': user_id,
            'session_id': session_id,
            'email': session.get('email', ''),
            'subscription_tier': session.get('subscription_tier', 'free')
        }
        
        return policy
        
    except jwt.ExpiredSignatureError:
        logger.warning("JWT token expired")
        raise Exception('Unauthorized - Token expired')
    except jwt.InvalidTokenError:
        logger.warning("Invalid JWT token")
        raise Exception('Unauthorized - Invalid token')
    except Exception as e:
        logger.error(f"Authorization error: {str(e)}")
        raise Exception('Unauthorized')

def verify_google_token(token):
    """Verify Google OAuth token and extract user information"""
    try:
        # Verify the token
        idinfo = id_token.verify_oauth2_token(token, requests.Request(), GOOGLE_CLIENT_ID)
        
        # Verify the issuer
        if idinfo['iss'] not in ['accounts.google.com', 'https://accounts.google.com']:
            raise ValueError('Wrong issuer.')
        
        # Extract user information
        return {
            'user_id': idinfo['sub'],
            'email': idinfo['email'],
            'name': idinfo['name'],
            'picture': idinfo.get('picture', ''),
            'email_verified': idinfo.get('email_verified', False)
        }
        
    except ValueError as e:
        logger.warning(f"Google token verification failed: {str(e)}")
        return None

def create_or_update_user(user_info):
    """Create new user or update existing user in DynamoDB"""
    
    user_data = {
        'user_id': user_info['user_id'],
        'email': user_info['email'],
        'name': user_info['name'],
        'picture': user_info['picture'],
        'email_verified': user_info['email_verified'],
        'updated_at': datetime.utcnow().isoformat(),
        'subscription_tier': 'free',
        'preferences': {
            'risk_tolerance': 'moderate',
            'investment_goals': [],
            'notifications_enabled': True
        }
    }
    
    # Check if user exists
    try:
        response = users_table.get_item(Key={'user_id': user_info['user_id']})
        if 'Item' in response:
            # User exists, update
            existing_user = response['Item']
            user_data['created_at'] = existing_user.get('created_at', user_data['updated_at'])
            user_data['subscription_tier'] = existing_user.get('subscription_tier', 'free')
            user_data['preferences'] = existing_user.get('preferences', user_data['preferences'])
        else:
            # New user
            user_data['created_at'] = user_data['updated_at']
    except Exception as e:
        logger.error(f"Error checking existing user: {str(e)}")
        user_data['created_at'] = user_data['updated_at']
    
    # Save user
    users_table.put_item(Item=user_data)
    
    return user_data

def create_user_session(user_id):
    """Create a new user session"""
    
    session_id = str(uuid.uuid4())
    expires_at = datetime.utcnow() + timedelta(days=7)  # 7 day expiration
    
    session_data = {
        'session_id': session_id,
        'user_id': user_id,
        'created_at': datetime.utcnow().isoformat(),
        'expires_at': int(expires_at.timestamp()),  # Unix timestamp for TTL
        'last_activity': datetime.utcnow().isoformat()
    }
    
    sessions_table.put_item(Item=session_data)
    
    return session_data

def verify_session(session_id, user_id):
    """Verify that a session is valid and not expired"""
    
    try:
        response = sessions_table.get_item(
            Key={'session_id': session_id}
        )
        
        if 'Item' not in response:
            return None
        
        session = response['Item']
        
        # Check if session belongs to user
        if session['user_id'] != user_id:
            return None
        
        # Check if session is expired (additional check beyond TTL)
        if session['expires_at'] < int(time.time()):
            return None
        
        # Update last activity
        sessions_table.update_item(
            Key={'session_id': session_id},
            UpdateExpression='SET last_activity = :activity',
            ExpressionAttributeValues={
                ':activity': datetime.utcnow().isoformat()
            }
        )
        
        return session
        
    except Exception as e:
        logger.error(f"Error verifying session: {str(e)}")
        return None

def generate_jwt_token(user_id, session_id):
    """Generate JWT token for authenticated user"""
    
    payload = {
        'user_id': user_id,
        'session_id': session_id,
        'iat': datetime.utcnow(),
        'exp': datetime.utcnow() + timedelta(days=7)
    }
    
    return jwt.encode(payload, JWT_SECRET, algorithm='HS256')

def generate_policy(principal_id, effect, resource):
    """Generate API Gateway policy for custom authorizer"""
    
    auth_response = {
        'principalId': principal_id
    }
    
    if effect and resource:
        policy_document = {
            'Version': '2012-10-17',
            'Statement': [
                {
                    'Action': 'execute-api:Invoke',
                    'Effect': effect,
                    'Resource': resource
                }
            ]
        }
        auth_response['policyDocument'] = policy_document
    
    return auth_response
```

### D.1 React/Next.js Implementation

#### Frontend Authentication Hook (useAuth.js)
```javascript
import { useState, useEffect, createContext, useContext } from 'react';

const AuthContext = createContext();

export const useAuth = () => {
  const context = useContext(AuthContext);
  if (!context) {
    throw new Error('useAuth must be used within an AuthProvider');
  }
  return context;
};

export const AuthProvider = ({ children }) => {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(true);
  const [token, setToken] = useState(null);

  useEffect(() => {
    // Check for existing token on mount
    const savedToken = localStorage.getItem('authToken');
    const savedUser = localStorage.getItem('userData');
    
    if (savedToken && savedUser) {
      setToken(savedToken);
      setUser(JSON.parse(savedUser));
    }
    setLoading(false);
  }, []);

  const signInWithGoogle = async (googleToken) => {
    try {
      setLoading(true);
      
      const response = await fetch(`${process.env.NEXT_PUBLIC_API_URL}/auth/google`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({ token: googleToken }),
      });

      if (!response.ok) {
        throw new Error('Authentication failed');
      }

      const data = await response.json();
      
      // Store auth data
      localStorage.setItem('authToken', data.token);
      localStorage.setItem('userData', JSON.stringify(data.user));
      
      setToken(data.token);
      setUser(data.user);
      
      return { success: true, user: data.user };
      
    } catch (error) {
      console.error('Sign in error:', error);
      return { success: false, error: error.message };
    } finally {
      setLoading(false);
    }
  };

  const signOut = () => {
    localStorage.removeItem('authToken');
    localStorage.removeItem('userData');
    setToken(null);
    setUser(null);
  };

  const isAuthenticated = () => {
    return !!token && !!user;
  };

  const getAuthHeaders = () => {
    return token ? { Authorization: `Bearer ${token}` } : {};
  };

  const value = {
    user,
    token,
    loading,
    signInWithGoogle,
    signOut,
    isAuthenticated,
    getAuthHeaders,
  };

  return (
    <AuthContext.Provider value={value}>
      {children}
    </AuthContext.Provider>
  );
};
```

```javascript
  const { signInWithGoogle } = useAuth();

  useEffect(() => {
    // Load Google Sign-In API
    const script = document.createElement('script');
    script.src = 'https://accounts.google.com/gsi/client';
    script.async = true;
    script.defer = true;
    document.body.appendChild(script);

    script.onload = () => {
      window.google.accounts.id.initialize({
        client_id: process.env.NEXT_PUBLIC_GOOGLE_CLIENT_ID,
        callback: handleGoogleSignIn,
      });

      window.google.accounts.id.renderButton(
        document.getElementById('google-signin-button'),
        {
          theme: 'outline',
          size: 'large',
          text: 'signin_with',
          shape: 'rectangular',
        }
      );
    };

    return () => {
      document.body.removeChild(script);
    };
  }, []);

  const handleGoogleSignIn = async (response) => {
    setIsLoading(true);
    
    try {
      const result = await signInWithGoogle(response.credential);
      
      if (result.success) {
        // Redirect to dashboard or main app
        window.location.href = '/dashboard';
      } else {
        console.error('Sign in failed:', result.error);
        // Show error message to user
      }
    } catch (error) {
      console.error('Sign in error:', error);
    } finally {
      setIsLoading(false);
    }
  };

  return (
    <div className="google-signin-container">
      <div id="google-signin-button" />
      {isLoading && (
        <div className="loading-overlay">
          <div>Signing in...</div>
        </div>
      )}
    </div>
  );
};

export default GoogleSignIn;
```

```javascript
const ProtectedRoute = ({ children }) => {
  const { isAuthenticated, loading } = useAuth();
  const router = useRouter();

      </div>
```

#### API Client with Authentication (apiClient.js)
```javascript
import { useAuth } from './useAuth';

class ApiClient {
  constructor(baseURL) {
    this.baseURL = baseURL;
  }

  async request(endpoint, options = {}) {
    const url = `${this.baseURL}${endpoint}`;
    
    const config = {
      headers: {
        'Content-Type': 'application/json',
        ...options.headers,
      },
      ...options,
    };

    try {
      const response = await fetch(url, config);
      
      if (!response.ok) {
        if (response.status === 401) {
          // Token expired, redirect to login
          localStorage.removeItem('authToken');
          localStorage.removeItem('userData');
          window.location.href = '/login';
          return;
        }
        throw new Error(`HTTP error! status: ${response.status}`);
      }

      return await response.json();
    } catch (error) {
      console.error('API request failed:', error);
      throw error;
    }
  }

  async get(endpoint, headers = {}) {
    return this.request(endpoint, {
      method: 'GET',
      headers,
    });
  }

  async post(endpoint, data, headers = {}) {
    return this.request(endpoint, {
      method: 'POST',
      headers,
      body: JSON.stringify(data),
    });
  }
}

// Hook for authenticated API calls
export const useApiClient = () => {
  const { getAuthHeaders } = useAuth();
  
  const client = new ApiClient(process.env.NEXT_PUBLIC_API_URL);
  
  return {
    get: (endpoint) => client.get(endpoint, getAuthHeaders()),
    post: (endpoint, data) => client.post(endpoint, data, getAuthHeaders()),
    // Add other methods as needed
  };
};

export default ApiClient;
```

## Troubleshooting Guide

### Common Issues

#### Issue 1: Google Token Verification Fails
```
Error: "Wrong issuer" or "Invalid token"

Solutions:
1. Check Google Client ID matches frontend configuration
2. Verify token is not expired
3. Ensure proper HTTPS configuration
4. Check Google API console settings
```

#### Issue 2: JWT Token Generation Fails
```
Error: "Key error" or "Encoding error"

Solutions:
1. Verify JWT_SECRET environment variable is set
2. Check JWT library version compatibility
3. Ensure proper datetime handling for exp claim
4. Validate payload structure
```

#### Issue 3: DynamoDB Access Denied
```
Error: "AccessDeniedException"

Solutions:
1. Check IAM role has proper DynamoDB permissions
2. Verify table names in environment variables
3. Ensure Lambda execution role is attached
4. Check table exists in correct region
```

#### Issue 4: API Gateway CORS Issues
```
Error: "CORS policy" or "preflight request"

Solutions:
1. Implement OPTIONS method for all endpoints
2. Set proper CORS headers in Lambda response
3. Check Access-Control-Allow-Origin header
4. Verify preflight response status is 200
```

### Debugging Steps
1. Check CloudWatch logs for Lambda function
2. Test Google token with Google's tokeninfo endpoint
3. Verify API Gateway logs for request/response details
4. Use AWS X-Ray for distributed tracing
5. Test with curl/Postman before frontend integration
